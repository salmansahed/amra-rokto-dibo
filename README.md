# 🩸 Amra Rokto Dibo

A modern blood donor management platform that connects blood donors with people in need of blood across Bangladesh. The platform makes it easier to find blood donors, register as a donor, and contribute to saving lives through a simple and user-friendly digital experience.

## 🌐 Live Demo

https://amra-rokto-dibo.vercel.app/

---

## 📖 Overview

**Amra Rokto Dibo** is a web-based blood donation platform designed to reduce the time and effort required to find blood donors during emergencies. Users can register as blood donors, search for available donors, and connect with them quickly.

The project aims to create a reliable and accessible blood donor network that helps save lives through community participation.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/amra-rokto-dibo.git
```

### Navigate to Project Directory

```bash
cd amra-rokto-dibo
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file in the root directory and add:

```env
MONGODB_URI=your_mongodb_connection_string

BETTER_AUTH_SECRET=your_secret_key
BETTER_AUTH_URL=http://localhost:3000
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
